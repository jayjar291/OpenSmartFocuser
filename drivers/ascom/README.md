# ASCOM Driver

Windows driver for OpenSmartFocuser using the [ASCOM Platform](https://ascom-standards.org).

## Requirements
- ASCOM Platform 6.6+
- .NET Framework 4.8+
- Visual Studio 2022 (to build from source)

## Install
Download the installer from the [Releases](../../releases) page.

## Build from Source
Open `OpenSmartFocuser.sln` in Visual Studio and build in Release mode.
